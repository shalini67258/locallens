## The Problem

When a power cut hits your street, when a road floods near your home, when an accident blocks your daily route — **where do you go for information?**

- WhatsApp groups are chaotic and unverified
- News channels only cover city-level events
- Government portals are slow and one-directional

**1.4 billion Indians have no reliable, real-time source for street-level alerts.**

---

## The Solution

**LocalLens** is a crowd-sourced, AI-verified hyperlocal alert network. Citizens report issues in their neighborhood. Every post is instantly analyzed by AI for credibility and urgency, then verified by the local community through voting and comments — creating a **multi-layer trust system** that's far more reliable than WhatsApp rumors or unverified posts.

---

## What Makes It Unique

| Other Platforms | LocalLens |
|----------------|-----------|
| Anyone can post fake news | AI detects and flags suspicious posts automatically |
| One report = unknown reliability | Multiple reporters merge into one verified card |
| English only | 22 Indian languages + voice posting in 10 languages |
| No civic intelligence | Government dashboard with real complaint analytics |
| City-level news only | Street-level, neighborhood-specific alerts |
| Manual verification | AI + community voting + comment sentiment = 3-layer trust |

---

## How It Works

```
Citizen reports issue
        │
        ▼
AI analyzes it (Groq LLaMA 3.3 70B)
  → Credibility: Likely Real / Suspicious
  → Severity:    High / Medium / Low
        │
        ▼
Community verifies it
  → 👍 Confirmed by locals
  → 👎 Denied by locals
  → 💬 Comments: Supports / Disputes / Neutral (AI-analyzed)
        │
        ▼
If 3+ people report same thing → Merged into one card
  → "4 people reported this — very likely happening!"
        │
        ▼
Area summary generated by AI
  → "Today in Kukatpally: Power cut since 9AM,
     heavy traffic near metro, water disruption tomorrow."
```

## Key Features

- 🤖 **AI Area Summary** — Groq AI reads all posts and generates a live area briefing
- 🛡️ **Fake News Detection** — every post rated ✅ Likely Real / ⚠️ Less Likely / 🚩 Suspicious
- 🔴 **Severity Scoring** — AI auto-tags posts as High / Medium / Low priority
- 🔁 **Duplicate Merging** — multiple reports of same incident merge into one verified card
- 💬 **Comment Sentiment** — AI analyzes each comment as Supports / Disputes / Neutral
- 🎙️ **Voice Post** — speak in any Indian language, auto-translated to English
- 🚨 **Emergency Mode** — one-tap filter for critical alerts only
- 🏛️ **Government Dashboard** — municipal analytics with area and category breakdown
- 🌐 **22 Indian Languages** — full page translation via Google Translate
- 📧 **Real Email Alerts** — Gmail SMTP welcome and area notifications
- 📄 **PDF Export** — download area reports as PDF
- 🛡️ **Trust Score** — user reputation system based on posts and community upvotes

## Tech Stack

| Layer | Technologies |
|-------|-------------|
| Frontend | React.js 18, Tailwind CSS, React Router v6, Recharts, jsPDF |
| Backend | Spring Boot 3.5, Spring Security, JWT Auth, JPA/Hibernate, Gmail SMTP |
| AI | Groq API — LLaMA 3.3 70B (credibility, severity, summary, sentiment) |
| Database | PostgreSQL (Render managed, persistent) |
| Free APIs | OpenStreetMap Nominatim, MyMemory Translation, Web Speech API |
| Deployment | Render — frontend + backend + database (all free tier) |

## Local Setup

```bash
# Backend
cd backend
# Configure application.properties with DB URL, Groq API key, Gmail credentials
mvn spring-boot:run        # runs at localhost:8080

# Frontend
cd frontend
npm install && npm start   # runs at localhost:3000
```
